Показывать код CSS я не стал, потому как всё зависит от выбранных тобой тегов (id и class и других). Тут всё довольно просто: если нужно употребить тег id, то в CSS начинаем запись с #. (Пример: #table) и перечисляем дальше в { }. Если же нам нужно употребить селектор class, то тут немного иначе: начинать нужно с . (Пример: .table) и перечисляем так же в { }.

И так, чем же они отличаются?

ID - уникальное собственное имя элемента на странице, то есть на странице не должно быть нескольких элементов с одним id.
CLASS - вольный признак, который дается обычно нескольким элементам, чтобы отличать их от других. Например, картинкам, которые просто сопровождают текст, можно дать (class="decor"), а картинкам, которые важны для понимания текста, — (class="content").

Часто используются такие свойства как padding (внутренний отступ) и margin (внешний отступ). Они бывают 4-х типов (-top -right -bottom -left), где top-вверх, right-вправо, bottom-вниз, left-влево. Вот пример:

.table2{
font-size: 15px;
color: #000000;
background-color: #dfebef;
border-radius: 10px;
margin-top: 20px;
margin-bottom: 20px;
padding-left: 50px;
padding-top: 10px;
padding-bottom: 10px;
}
В данном случае (font-size, color, background-color...) - Свойства, а (15px, #000000, #dfebef...) - Значения.

Какими бывают свойства?

Самыми популярными являются font-size, color, background-color, margin, padding, border, border-radius, font-family, width, height ну и пожалуй всё.

Теперь рассмотрим по-подробнее, за что отвечает каждый из них.

font-size: 16px;

<-- Отвечает за размер чего-либо, где 16px - его величина -->

color: #000000;

<-- Отвечает за цвет чего-либо, где #000000 - его цвет -->

background-color: #000000;

<-- Отвечает за цвет фона, где #000000 - его цвет -->

margin: 16px;

<-- Отвечает за наружний отступ, где 16px - его величина -->

padding: 16px;

<-- Отвечает за внутренний отступ, где 16px - его величина -->

border: 16px;

<-- Отвечает за размер рамки, где 16px - его величина -->

border-radius: 16px;

<-- Отвечает за степень закруглённости угла, где 16px - его величина -->

font-family: Montserrat;

<-- Отвечает за шрифт, где Montserrat - его название -->

width: 16px;

<-- Отвечает за ширину объекта, где 16px - его величина -->

height: 16px;

<-- Отвечает за высоту объекта, где 16px - его величина -->

font-weight: 16px;

<-- Отвечает за толщену объекта, где 16px - его величина -->

Следующее с чем я столкнулся - вёрстка сайта и шрифт

Как можно скачать и установить шрифты, и как добавить (шапку, меню, блок и подвал) на сайт я сейчас расскажу. Сперва стоит найти подходящий шрифт, к счастью сайтов для этого полно, затем скачать расширение Typ.io (по ссылке ниже или в блоке). Как только всё сделано можно начинать.

Открываем Typ.io и жмём на +, после чего щёлкаем на текст, шрифт которого тебе приглянулся. В окошке сверху справа отобразится текст "You're looking at ...". Три точки и будут названием шрифта.

Чтобы скачать шрифт идём в интернет и пишем "Скачать шрифт ...". Открываем сайты, в которых будет расширение woff или woff2. Если будут два, то качаем два. Это нужно для того, чтобы пользователь, даже на самом древнем браузере, смог увидеть шрифт установленный на сайте. Для привязки шрифта к сайту используем:
body{
margin-left:100px;
margin-right:100px;
font-family:Montserrat;
}

Как найти подходящий цвет?

Для того чтобы не выдумывать самому палитру цветов, так же заходим на сайты и ищем понравившиеся цвета. Далее скачиваем EYE DROPPER - для определения цвета или устанавливаем расширение для браузера (по ссылке внизу экрана). Клацаем на кнопку Pick color from web page и, наведя курсор мыши на нужный цвет, нажимаем. EYE DROPPER покажет тебе номер цвета (Пример: #000000 - black), которым можно будет использовать в CSS.

На этом с цветом всё. Цвет является большой составляющей CSS и знать как он работает очень полезно.

Что нужно знать о JavaScript?

Для начала познакомимся с jQuery. jQuery - это специальная библиотека, написанная на JavaScript, в которой многие типовые действия прописаны как одна команда. Представь, что это набор готовых деталей конструктора, из которого собрать программу получается быстрее. Правда, не все задачи решаются деталями конструктора…

Чтобы было понятно, насколько jQuery может ускорить написание кода, вот пример. Двум программистам дали задачу: сделать так, чтобы по клику в самом низу страницы она сама плавно прокручивалась до самого верха. Один решил эту задачу на чистом JavaScript:

var t; function scrolltop() {
var top = Math.max(document.body.scrollTop,document.documentElement.scrollTop); if(top > 0 ) {
window.scrollTo(0,Math.floor(top/1.5)); t = setTimeout("scrolltop()",30);
} else {
clearTimeout(t);
} return false;
}

Другой использовал jQuery. У него код состоит из одной строки:

$("html,body").animate({scrollTop:0},500);

У обоих программистов код работает одинаково, но второй решил задачу гораздо быстрее.

Как включить jQuery?

Чтобы вся магия jQuery заработала, нужно его подключить в HTML-странице:
<script src=”https://yastatic.net/jquery/3.3.1/jquery.min.js ” type=”text/javascript”></script >

Эта команда говорит браузеру: «Иди на сервер Яндекса по вот такому пути, найди там файл jquery.min.js, будем работать с этой библиотекой и писать на ней скрипты».

Простой доступ к странице

В чистом JavaScript обратиться к объекту на странице можно одним из способов:

document.getElementById("myElement")
document.getElementsByTagName("td")
document.getElementsByClassName("myClass")

и еще несколько подобных

В jQuery то же самое делает одна прекрасная команда. Заодно она же совмещает в себе кучу других функций. Фактически она говорит jQuery: «Возьми ЭТО и сделай с ним что-то»:

$("myElement")

<-- Вместо знака $ можно использовать слово jQuery -->

Поэтому какие-то простые манипуляции с объектами на странице выглядят элегантнее именно с jQuery. Это не значит, что их нельзя делать с простым JavaScript — можно. Но это не так красиво.
